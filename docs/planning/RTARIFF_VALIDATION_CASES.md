# R-Тариф Validation Matrix — универсальная сертификация ТР-1 2026

> Сгенерировано из `scripts/seed-data/rtariff-validation.template.json`. Не редактируйте таблицы вручную — правьте JSON и пересоберите. Цель: подтвердить расчёт «в рубль» по всему универсальному пространству {плечо}×{вагон}×{принадлежность}×{отправка}×{класс 1/2/3}×{маршруты РФ + узлы Москва/СПб + СНГ}.

## Как пользоваться

1. Откройте R-Тариф Онлайн, выставьте дату расчёта = `Дата` из карточки.
2. Введите станции **по ЕСР** (имена — подсказка, ЕСР — источник истины).
3. Задайте ЕТСНГ, тип вагона, принадлежность, вид отправки (+кол-во вагонов), массу на вагон.
4. Для инновационных кейсов укажите точную модель; иначе «кроме перечисленных» (классик).
5. Впишите ответ R-Тарифа в **пустые поля** карточки: провозная плата (без НДС) / с НДС / км.
6. Перенесите те же числа в JSON (`ref_distanceKm`, `ref_perWagonProvoznaya_noVat`, `ref_perWagonWithVat`) и запустите `npx tsx scripts/validate-rtariff.ts` — он выдаст diff-таблицу (км Δ / ₽ Δ / verdict).

## Покрытие движка (легенда)

| coverage | смысл | ожидание |
|---|---|---|
| **proven** | own-ПВ class-1, путь N8 — сертифицирован на 3 оракулах | МАТЧ до ₽0 |
| **fitted** | own-ПВ class-1, но проверяет один из 3 подогнанных рычагов вне точки подгонки | ожидаем МАТЧ; расхождение = рычаг неверен |
| **gap** | rzd В-компонента / крытый/платформа/цистерна/хоппер / класс 2,3 — движок ещё не моделирует | RED или приблизительно; собираем эталон для постройки |

## 3 подогнанных рычага (приоритет проверки)

- **SHORT_HAUL_BOUNDARY_UPLIFT** — множитель K4 на ≤2000 км, подогнан под ЕДИНСТВЕННЫЙ оракул (ЭТ201459, 699 км). Кейсы: C03, C06, C09, C12, C15, C36, C38 (разные кол-ва вагонов × короткие/средние пояса) сертифицируют или опровергают.
- **C_OWN_PV_CLASS1 = 0.9346** — сценарный коэф. own-ПВ class-1. Кейсы: C01, C02, C04, C07, C10, C33, C37, C39 (разные грузы класса 1, плечи) проверяют обобщаемость.
- **C_INNOVATIVE = 0.9595** — инновационный полувагон. Пара C13 (классик) / C14 (иннов., та же трасса) изолирует рычаг: отношение C14/C13 в R-Тарифе должно быть ровно 0.9595.
