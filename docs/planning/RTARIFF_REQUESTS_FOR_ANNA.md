# Задание для Анны — эталонные расчёты R-Тариф

> **Зачем.** Мы строим свой калькулятор тарифа РЖД (ТР-1 2026), который должен считать
> провозную плату **до копейки** на всех родах вагонов, контейнерах и классах груза.
> Чтобы доверять цифре, каждый сценарий нужно один раз сверить с R-Тарифом (он у нас как эталон).
> Уже сверены до рубля: **собственный полувагон, класс 1, щебень/нерудные**. Остальное —
> считается, но пока не подтверждено. Твои расчёты ниже закрывают это.

## Как присылать (важно)

По **каждому** расчёту — **скриншот из R-Тарифа целиком**, чтобы было видно:
- маршрут и **расстояние, км**;
- схему/тариф и коэффициенты, которые R-Тариф применил;
- **провозную плату за вагон без НДС и с НДС**, и **за тонну**;
- дату расчёта и что это **ТР-1 (2026)**.

Скриншот важнее, чем переписать цифры руками — на нём видно, как R-Тариф пришёл к результату.

## БЛОК 1 — СРОЧНО (сегодняшнее расхождение) ⚡

**Маршрут: Элисенваара → Решетниково.**

Посчитай в R-Тарифе **ровно с теми же параметрами**, что мы считали сегодня, и пришли:

1. **Расстояние, км** по R-Тарифу.
2. **Через какой узел/ход** идёт маршрут (если R-Тариф показывает трассу — Москва / в обход).
3. Параметры расчёта: груз (ЕТСНГ + класс), род вагона, собственный/общий парк, г/п и масса груза, отправка (повагонная/групповая), число вагонов.
4. **Провозная плата за вагон без НДС и с НДС**, за тонну.
5. **Наша цифра vs цифра R-Тарифа** — отдельно **километры** и отдельно **рубли** (что именно разошлось).

> Подозрение: расхождение в **расстоянии** из-за московской узловой надбавки (Решетниково стоит за
> Ховрино — это Московский узел). Поэтому особенно нужен точный **км** от R-Тарифа.

## БЛОК 3 — «тариф предоставления» (главное, что мы не добили) ⭐

Нужен расчёт по **ОБЩЕМУ (инвентарному) парку** — это тариф с **вагонной составляющей**, на основе
которого считается ставка предоставления.

**Параметры:** общий (инвентарный) парк, **полувагон**, щебень **класс 1**, отправка **групповая** и
**повагонная**, расстояние **~1500 км** (любой реальный маршрут в этом диапазоне).

Критично — попроси R-Тариф показать **разбивку по составляющим**:

| Составляющая | Групповая | Повагонная |
|--------------|-----------|------------|
| **Инфраструктурная + локомотивная (И)** | | |
| **Вагонная составляющая (В)** | | |
| **Итого провозная без НДС** | | |
| С НДС | | |
| Расстояние, км | | |

> Без отдельной строки «вагонная составляющая (В)» мы не можем сверить инвентарный тариф —
> а на нём держится вся ставка предоставления.
