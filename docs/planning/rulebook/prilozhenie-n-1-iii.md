## 0. TL;DR — что это и КАК входит в расчёт

Раздел III — это **не грузовой тариф за перевозку**. Это **отдельный тариф «инфраструктурного перевозчика»**: его платит участник, который сам организует пробег грузового поезда (или одиночного локомотива) по путям РЖД своим подвижным составом/локомотивами. Он считается по своим тарифным схемам: **ЗИ N 1–3** (занятие инфраструктуры), **ОПВ N 1–3** (продвижение вагонов/единиц подвижного состава) и **ОПЛ N 1–2** (продвижение локомотивов).

**Это НЕ та формула, что в движке SimpleCargo.** Сертифицированная формула движка (`scripts/seed-data/reference-quotes-rtariff.json`) построена на **Разделе II** — грузовом тарифе (Схема 8/8(1), И1, K1, схема 25(1) для порожнего). Раздел III — параллельная ветка для случая «перевозчик не РЖД» (поездное формирование, не принадлежащее РЖД). Для текущего продукта (экспедитор, считает провозную плату клиенту по общему парку) Раздел III **не применяется** и его коэффициенты в сидах отсутствовать — это нормально, **не баг**.

Структура расчёта по Разделу III (п.52.5): тариф складывается из ТРЁХ независимо считаемых и независимо округляемых частей (п.50.2, п.50.8):
1. **Занятие инфраструктуры** грузовым поездом / одиночным локомотивом → схемы **ЗИ N 1–3**.
2. **Продвижение единиц подвижного состава** (вагоны и пр.) в составе поездов → схемы **ОПВ N 1–3**.
3. **Продвижение локомотива** (тяга / одиночное следование) → схемы **ОПЛ N 1–2**.

## 3. Plain-Russian: КАК это входит в расчёт (если бы движок считал Раздел III)

Для каждого пробега тариф собирается из трёх слагаемых (п.52.5), каждое округляется отдельно (п.50.8):

1. **Шаг ЗИ (занятие инфраструктуры, п.53):** `ЗИ N 1` (начально-конечная операция, не зависит от км) + `ЗИ N 2` × км по электрифицированному участку + `ЗИ N 3` × км по неэлектрифицированному. Множители: ×0,91 (п.53.2 — без сортировочного парка) или ×0,14 (п.53.3 — одиночный локомотив / короткий пробег ≤50 км).
2. **Шаг ОПВ (продвижение вагонов, п.54):** за каждую единицу ПС по `ОПВ N 1` (гружёный, за массу но не менее МВН/10/22 т) или `ОПВ N 2` (спецвагоны/транспортёры) или `ОПВ N 3` (порожние). Базовая ставка × коэффициент длины/массы тары (табл. N 33–35) × коэффициент класса/рода/расстояния (табл. N 4 и N 36). Для цистерн дополнительно × {3 | 2 | 2,2 | 1,6} по тарифному классу (п.54.1.2). Спецпороги: ×0,66 и ×0,42 (>1200 км) для порожних спецвагонов после контейнеров (п.54.2); ×0,47 (>4000 км) для контейнерных вагонов (п.54.3); коэффициенты Калининградской ж.д. (п.54.4, табл. N 3).
3. **Шаг ОПЛ (продвижение локомотива, п.55):** за каждую секцию по `ОПЛ N 1` (электровоз) или `ОПЛ N 2` (тепловоз). ×0,78 (п.55.2 — 3-я+ секция / доп. тяга / подталкивание); ×0,43 (п.55.3 — электровоз одиночным следованием).

Поверх всего — общий ×0,9116 (п.50.1), если РЖД оплачивает использование чужой инфраструктуры.

**Единица расстояния** — км по согласованному маршруту (п.4 ТР), причём электрифицированные и неэлектрифицированные участки суммируются и тарифицируются раздельно (п.51.1). Округление длины/тары/грузоподъёмности — п.50.6; массы груза — п.50.7 (через п.15 ТР); итогов — п.50.8 (через п.15 ТР).
