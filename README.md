# SimpleCargo

PWA для экспедитора железнодорожных вагонов **ООО «РНС»**. Берём полувагоны у собственников по фиксированной
цене и перепродаём перевозку клиентам; прибыль = маржа. Долгосрочная цель — AI-агенты на почте, которые
автоматически ведут отчётность по дислокациям и поручениям.

> ⚙️ Статус: фаза планирования завершена. Код приложения ещё не написан (Фаза 0 — далее).

Railway (GitHub CI/CD) · PWA.

## Бизнес-модель (проверено на реальных данных)

- **Маржа** = `Сумма УА` (выручка с клиента) − `Сумма от Поставщика` (себестоимость от собственника)
- **Оборот, сут** = полный цикл вагона (загрузка → рейс → выгрузка → возврат → след. загрузка)
- Цены приходят из **ПСЦ** (тариф-таблица по маршрутам) и **заявок** (поручения к договору)
